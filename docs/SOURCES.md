# SOURCES — официальные источники по теме дронов

Этот файл — реестр ссылок, которые бот использует как источники правды для юридических и регуляторных разделов.

**Правила работы с этим файлом:**

- Только официальные сайты и нормативные документы. Никаких блогов, статей и форумов в качестве источника закона.
- Каждая запись: название, краткое описание, URL, дата проверки.
- При сомнениях — лучше пропустить ссылку, чем поставить непроверенную.
- Если ссылка устарела или сайт переехал — обновить и проставить новую дату проверки.
