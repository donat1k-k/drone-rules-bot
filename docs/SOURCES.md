# SOURCES — официальные источники по теме дронов

Этот файл — реестр ссылок, которые бот использует как источники правды для юридических и регуляторных разделов.

**Правила работы с этим файлом:**

- Только официальные сайты и нормативные документы. Никаких блогов, статей и форумов в качестве источника закона.
- Каждая запись: название, краткое описание, URL, статус проверки.
- При сомнениях — лучше пропустить ссылку, чем поставить непроверенную.
- Пометка «требует ручной проверки» означает, что URL указан предположительно и должен быть проверен живым человеком перед защитой/деплоем.

---

## 1. Регистрация БВС (беспилотных воздушных судов)

- **Название:** Госуслуги — Государственный учёт БВС
- **Описание:** Официальный портал для постановки беспилотного воздушного судна на государственный учёт в России.
- **URL:** https://www.gosuslugi.ru (ищи услугу «Государственный учёт беспилотных воздушных судов»)
- **Статус:** ⚠️ Требует ручной проверки перед защитой — уточни актуальный прямой URL услуги.

## 2. Закон / нормативные акты по использованию дронов

- **Название:** Официальный интернет-портал правовой информации
- **Описание:** Полные тексты федеральных законов и постановлений Правительства РФ, включая акты, регулирующие использование БВС. Ищи по запросу «беспилотные воздушные суда» или «БВС».
- **URL:** https://pravo.gov.ru
- **Статус:** ⚠️ Требует ручной проверки — открой портал и найди актуальные нормы самостоятельно.

- **Название:** КонсультантПлюс — Воздушный кодекс РФ
- **Описание:** Текст Воздушного кодекса Российской Федерации. Основной закон, регулирующий воздушное пространство и использование воздушных судов, в том числе беспилотных.
- **URL:** https://www.consultant.ru (ищи «Воздушный кодекс Российской Федерации»)
- **Статус:** ⚠️ Требует ручной проверки — убедись, что находишь актуальную редакцию.

> ℹ️ Конкретные номера статей и постановлений в тексте бота **не использовать** без проверки актуальной редакции закона.

- **URL:** https://favt.gov.ru
- **Статус:** ⚠️ Требует ручной проверки — найди актуальный раздел с картами воздушного пространства на сайте.

## 4. Получение разрешения на полёт / уведомление органов

- **Название:** Госуслуги — уведомление об использовании воздушного пространства
- **Описание:** Для ряда полётов требуется подача уведомления или плана полёта в органы организации воздушного движения (ОВД). Услуга может быть доступна через Госуслуги.
- **URL:** https://www.gosuslugi.ru (ищи услугу, связанную с уведомлением ОВД или использованием воздушного пространства)
- **Статус:** ⚠️ Требует ручной проверки — найди актуальную услугу на портале самостоятельно.

- **Название:** Росавиация — организация воздушного движения
- **Описание:** Официальная информация о порядке использования воздушного пространства и уведомлении органов ОВД.
- **URL:** https://favt.gov.ru
- **Статус:** ⚠️ Требует ручной проверки.

## 7. Метеоданные для подготовки к полёту

- **Название:** Windy
- **Описание:** Детализированный прогноз ветра, видимости и погодных условий на различных высотах. Используется как вспомогательный инструмент при подготовке к полёту.
- **URL:** https://www.windy.com
- **Статус:** Проверено — сайт существует (2026-05-23).

- **Название:** Open-Meteo
- **Описание:** Бесплатный открытый погодный API без ключа. Используется в боте для функции «🌦 Погода для полёта» — возвращает текущие данные по температуре, ветру и осадкам по координатам.
- **URL:** https://open-meteo.com
- **Статус:** Проверено — API работает (2026-05-24). Лицензия CC BY 4.0.

> ℹ️ Погодная оценка в боте **справочная** — не является разрешением на полёт. Лётные решения принимаются пилотом самостоятельно с учётом всех условий.

## Чеклист перед защитой/деплоем

1. Открыть каждый URL с пометкой «Требует ручной проверки» и убедиться, что сайт открывается.
2. Найти актуальные прямые ссылки для секций 1 и 4 (услуги на Госуслугах).
3. Сверить тексты бота с тем, что реально написано в источниках.
4. Не добавлять в бот номера статей, суммы штрафов и точные пороги без проверки здесь.
