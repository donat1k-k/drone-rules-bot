## Notes for Next Agent

- Запуск бота: `python -m src.bot` из корня проекта.
- Docker: `docker compose up -d --build`.
- Структура YAML: `id`, `title`, `body`, `disclaimer` (опц.), `keywords` (список строк). Не менять структуру без записи в DECISIONS.md.
- Токен и ключи — только в `.env`, не в коде. Никогда не коммить.
- AI-роутер и weather-роутер в `__main__.py` регистрируются **перед** common-роутером — иначе fallback перехватит FSM-сообщения.
- Open-Meteo не требует ключа — погода работает всегда.
- При смене провайдера AI — только `.env` (OPENAI_BASE_URL, OPENAI_MODEL, OPENAI_API_KEY), код менять не надо.
- Полная карта проекта для нового разработчика — [MAINTENANCE.md](MAINTENANCE.md).
