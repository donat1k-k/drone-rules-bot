# drone-rules-bot

Telegram-бот **«Правила управления дронами для начинающих»**.
Учебный проект 11 класса: дружелюбный гид по подготовке, безопасности и юридическим рамкам полётов на дроне для новичков.

> 📚 Учебный проект. Готов к демонстрации.
> ⚠️ Бот предоставляет **справочную** информацию, а не юридическую консультацию.

## Как работает бот

1. Пользователь открывает диалог, отправляет `/start`.
2. Бот показывает inline-меню из 8 разделов + дополнительных функций.
3. Любую кнопку можно нажать → бот присылает текст раздела + кнопку «← Главное меню».
4. Вместо меню можно **написать вопрос текстом** — бот ищет в базе по ключевым словам.
5. Если ничего не нашлось — fallback с кнопкой «✍️ Оставить вопрос».
6. При включённом AI — кнопка «🤖 Спросить ИИ» отвечает на свободные вопросы с опорой на базу знаний.

Архитектурно бот делится на изолированные слои:

- **Ядро** (меню + база знаний) — работает всегда.
- **AI-слой** — подключается при наличии ключа, отсутствие не ломает бот.
- **Погода** — внешний API без ключа.

## AI-режим

Бот поддерживает любой **OpenAI-compatible API** — OpenAI, Polza AI, прочие совместимые провайдеры.

**Что делает:**
- Отвечает на свободные вопросы про дроны.
- Получает базу знаний бота как контекст (заголовки + ключевые слова всех 8 разделов).
- **Не выдумывает законы и штрафы.** По юридическим темам отсылает к Госуслугам и Росавиации.
- В конце каждого ответа — дисклеймер о справочном характере.
- При падении провайдера возвращает «попробуй позже» — остальной бот работает.

**Включение** — две переменные в `.env`:

```env
AI_ENABLED=true
OPENAI_API_KEY=<твой ключ>
```

При `AI_ENABLED=false` или пустом ключе кнопка «🤖 Спросить ИИ» **не показывается** в меню. Подробности — [docs/ENVIRONMENT.md](docs/ENVIRONMENT.md).

Логика — [`src/ai/client.py`](src/ai/client.py).

## Погода для полёта

Кнопка **«🌦 Погода для полёта»** доступна в главном меню без настройки — ключ не нужен.

- Бэкенд: [Open-Meteo](https://open-meteo.com) (бесплатно, без ключа, лицензия CC BY 4.0).
- Геокодинг: Open-Meteo Geocoding API (тот же провайдер).
- Ввод: **название города** (например, `Москва`) или **координаты** в формате `55.75, 37.61`.

Бот возвращает температуру, скорость ветра, осадки и светофор-оценку:

- ✅ — спокойные условия для тренировочного полёта
- ⚠️ — требуется осторожность (ветер 5–9.9 м/с)
- ⛔ — лучше отложить (ветер ≥10 м/с или осадки)

> Оценка **справочная** и не является разрешением на полёт.

Логика — [`src/weather/client.py`](src/weather/client.py).

## Мини-тест

Кнопка **«🧠 Мини-тест»**: 5 inline-вопросов по безопасности, регистрации, запретным зонам и Return-to-Home.

- После каждого ответа — ✅ или ❌ + короткое объяснение.
- В конце — итоговый счёт `X/5` и кнопка «← Главное меню».

Реализация — [`src/bot/handlers/quiz.py`](src/bot/handlers/quiz.py).

## Обратная связь

Если бот не нашёл подходящий раздел, в fallback-ответе появляется кнопка **«✍️ Оставить вопрос»**.

- Бот переходит в FSM-режим ожидания текста.
- Пользователь пишет вопрос → бот отвечает «✅ Вопрос сохранён».
- Сохраняется только **текст + ISO-timestamp** в `logs/feedback.txt`.
- `user_id`, `username`, `chat_id` **не сохраняются** — персональных данных в файле нет.
- Папка `logs/` — в `.gitignore`.

Реализация — [`src/bot/handlers/feedback.py`](src/bot/handlers/feedback.py).

## Быстрый запуск локально

**Требования:** Python 3.11+, токен от [@BotFather](https://t.me/BotFather).

```bash
# 1. Клонировать
git clone <url-репозитория>
cd drone-rules-bot

# 2. Виртуальное окружение + зависимости
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

pip install -r requirements.txt

# 3. Настроить .env
cp .env.example .env
# открой .env и подставь TELEGRAM_BOT_TOKEN

# 4. Запустить
python -m src.bot
```

Открой диалог с ботом в Telegram → `/start`.

## Пример `.env` для Polza AI

[Polza AI](https://polza.ai) — OpenAI-compatible провайдер, через который доступны разные модели одним ключом.

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here

AI_ENABLED=true
OPENAI_API_KEY=your_api_key_here
OPENAI_BASE_URL=https://polza.ai/api/v1
OPENAI_MODEL=google/gemini-3.1-flash-lite

LOG_LEVEL=INFO
```

> ⚠️ Подставь свой реальный ключ только в локальный `.env`. **Не коммить** ключи в репозиторий — `.env` в `.gitignore`.

## Запуск через Docker

**Требования:** Docker Desktop (Windows/Mac) или Docker Engine (Linux) с плагином `docker compose v2`.

```bash
# 1. Заполни .env (минимум TELEGRAM_BOT_TOKEN)
cp .env.example .env

# 2. Собрать образ и запустить в фоне
docker compose up -d --build

# 3. Логи
docker compose logs -f

# 4. Остановить
docker compose down
```

Бот работает через long polling — webhook и HTTPS-домен не нужны. `restart: unless-stopped` — контейнер поднимается автоматически после ребута.

## Smoke-тест (ручная проверка)

1. `/start` → приветствие + меню (включая «🌦 Погода» и «🧠 Мини-тест»).
2. Нажми каждую кнопку → текст раздела открывается.
3. «← Главное меню» → возврат в меню.
4. Напиши `нужно ли регистрировать дрон` → бот находит раздел регистрации.
5. Напиши `как купить пиццу` → fallback с кнопками «✍️ Оставить вопрос» и «← Главное меню».
6. «✍️ Оставить вопрос» → текст → «✅ Вопрос сохранён»; файл `logs/feedback.txt` создан.
7. `/help` → упоминание текстового ввода.
8. В разделах «Запрещённые зоны», «Регистрация», «FAQ» виден дисклеймер.
9. FAQ — 11 вопросов и ответов.
10. «🌦 Погода» → `Москва` → результат с оценкой и дисклеймером.
11. «🌦 Погода» → `55.75, 37.61` → результат по координатам.
12. «🌦 Погода» → `ываыва` → «Город не найден» → ввести `Краснодар` → результат **без возврата в меню**.
13. «🧠 Мини-тест» → 5 вопросов → итог X/5 → «← Главное меню».
14. *(при AI_ENABLED=true)* «🤖 Спросить ИИ» → `как откалибровать компас` → ответ с дисклеймером.

## Безопасность и секреты

- `.env` находится в [`.gitignore`](.gitignore) — токен в репозиторий не попадёт.
- `logs/` — в `.gitignore`, обратная связь хранится локально.
- `__pycache__`, `.venv` — тоже в `.gitignore`.
- В `.env.example`, README, docs и коде **нет реальных токенов** и API-ключей — только placeholder'ы.
- Если ключ случайно попал в git: **отозви его у провайдера**, замени, удали из истории.
- На VPS — `chmod 600 .env`.

## Юридический дисклеймер

Бот предоставляет **справочную информацию** по правилам полётов на дроне для начинающих.

- Бот **не является** юридической консультацией.
- Бот **не выдаёт разрешений** на полёт.
- Бот **не является** официальным источником законов, штрафов или карт запретных зон.
- Все юридические вопросы (регистрация, нормативы, штрафы, зоны) необходимо проверять на официальных источниках — Госуслуги, Росавиация, Воздушный кодекс РФ. См. [docs/SOURCES.md](docs/SOURCES.md).
- AI-режим обязан опираться на базу знаний и официальные ссылки. Системный промпт запрещает выдумывать законы.
