# drone-rules-bot

Учебный проект 11 класса: Telegram-бот **«Правила управления дронами для начинающих»**.

## Что это

Telegram-бот-инструктор, который объясняет новичкам:

- подготовку к полёту;
- правила безопасности;
- запрещённые зоны (общие принципы + ссылки на официальные карты);
- регистрацию и закон (справочно);
- что делать при потере сигнала;
- советы новичкам;
- FAQ;
- полезные ссылки;
- AI-режим для свободных вопросов (опционально).

## Дисклеймер

Бот предоставляет **справочную** информацию. Это **не юридическая консультация**.
Все вопросы по регистрации, законам и запретным зонам нужно сверять с официальными источниками — см. [docs/SOURCES.md](docs/SOURCES.md).

## Статус

Stage 0 — подготовлены документы и рамки проекта. Кода бота пока нет.
План работ: [docs/TODO.md](docs/TODO.md). Текущее состояние: [docs/HANDOFF.md](docs/HANDOFF.md).

## Технологии (план)

- Python 3.11+
- aiogram 3.x (Telegram Bot API)
- База знаний в YAML/JSON
- Конфигурация через `.env`
- AI-режим: подключение к внешнему LLM-API (опционально, бот работает и без него)

## Как запускать

Инструкция появится на Stage 1 после реализации базового бота. Сейчас запускать нечего.

Превью переменных окружения — в [.env.example](.env.example).
